Twin Terry Tao

2026

Tries to reconstruct how Terence Tao actually reasons through a problem, not just what he'd say. Multi-hop retrieval chases down supporting context across steps, hybrid dense + sparse search keeps recall high on niche math terminology, and a chain-of-thought loop forces the model to show its work before committing to an answer. Runs entirely on a self-hosted quantised model.

2026

Turns hand gestures into desktop actions. Each gesture fires a request to Groq, which writes and runs a Python script on the spot — so instead of a fixed set of pre-programmed commands, you can teach it new gestures and rewire what each one does whenever you want.
